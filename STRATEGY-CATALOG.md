# 트레이딩 전략 카탈로그 — 공개 전략 조사 & 검증 상태

> **목적:** 인터넷·학술·트레이딩 고전에 공개된 전략을 모아, 우리 백테스트 +
> 워크포워드 틀로 **직접 검증**한다.
>
> **대전제(중요):** 공개된 전략은 대부분 이미 엣지가 닳았거나(alpha decay),
> 과최적화·체리피킹된 "수익 자랑"이다. **수익률 주장은 전부 무시하고, 매매 규칙만
> 추출해서 직접 돌린다.** 닳은 것은 우리 검증에서 탈락, 살아남으면 후보로 격상.

## 전략별 상세 (규칙 = 검증에 쓸 핵심만)

### 1. 추세추종 — 이동평균 / 추세필터
- **규칙:** 가격 > 기준 이동평균(또는 단기MA>장기MA) → 보유, 아니면 현금.
- **특성:** 승률 낮고 큰 추세에서 번다. 강세장에서 단순보유에 못 미치나 하락 방어.
- **검증:** MA교차는 워크포워드 완패(3편). 추세필터(>MA200)는 워크포워드 통과(4편) —
  "비슷하게 벌고 덜 잃는다" 달성. **현재 실거래 후보 1순위.**

### 2. 모멘텀 — 절대 / 듀얼
- **절대 모멘텀:** 최근 N개월(보통 12개월) 수익률 > 0 → 보유, < 0 → 현금/채권.
- **듀얼 모멘텀(안토나치 GEM):** 절대 모멘텀으로 시장 방향 판단 + 여러 자산 중 상대강도
  1등 보유. 12개월 룩백, 월간 리밸런싱, 연 2~3회 거래.
- **특성:** 강세장엔 못 미치나 약세장 방어로 장기 우위, 낙폭이 벤치마크보다 작음
  (우리 추세필터와 **같은 철학**).
- **검증:** 절대 모멘텀(90일)은 4편에서 함 — 총수익 ↓, MDD ↓. 듀얼은 다자산 필요.

### 3. 변동성 돌파 (래리 윌리엄스) — ★ 다음 검증 1순위
- **핵심 아이디어:** 하루 변동폭이 크면 새 추세의 시작 신호.
- **규칙:** 오늘 시가 + k × (전일 고가 − 전일 저가) 를 가격이 **돌파하면 당일 매수**,
  **당일 종가에 청산**. k는 보통 0.5.
- **특성:** 단기(하루) 전략. 한국 퀀트 커뮤니티에서 가장 유명한 전략 중 하나.
- **왜 1순위:** BTC 일봉 OHLC로 바로 검증 가능하고, 우리가 아직 안 한 **완전히 다른
  계열**(일중 돌파)이라 비교 가치가 큼.

### 4. 평균회귀 — RSI / 볼린저밴드
- **RSI:** RSI < 30(과매도) 매수, > 70(과매수) 매도.
- **볼린저밴드:** 하단 밴드 터치 매수, 상단 매도(평균 회귀에 베팅).
- **특성:** 단기, 승률 높지만 큰 추세에 역행하면 크게 다침. 추세추종과 **정반대 철학**.
- **검증:** 미검증. 추세장(BTC)에서 평균회귀가 통할지 회의적이나, 직접 확인 가치 있음.

### 5. 코인 특화 — DCA / 그리드 / 차익 / 마틴게일
- **DCA(분할매수):** 정기적으로 일정액 매수. 단, 2024–2025 테스트에서 일부 BTC DCA가
  **단순보유에 못 미쳤다**(공개 데이터). → 우리 검증 가치 있음.
- **그리드:** 일정 간격마다 매수/매도 반복. **횡보장 전용**, 추세장에선 약함.
- **차익거래:** 거래소·체인 간 가격차. 다거래소 데이터·인프라 필요(우리 범위 밖).
- **⚠️ 마틴게일:** 손실 시 베팅을 2배로. **절대 권장 안 함** — 한 번의 큰 추세에 계좌
  전체가 날아갈 수 있음. 검증 대상에서 제외(위험성 경고용으로만 언급).

## 조사에서 확인된 '정직한' 사실들 (우리 결론과 일치)

- "약세장에서는 **어떤 봇도 손실을 막지 못한다.**" → 우리 1·3편 결론과 동일.
- BTC DCA조차 **단순보유에 못 미친 사례**가 공개 데이터로 존재. → 4편 결론과 동일.
- 듀얼 모멘텀도 **강세장엔 못 미치나 낙폭 방어로 장기 우위.** → 우리 추세필터와 동일 철학.
- "잘못된 설정은 **수동매매보다 더 빨리 잃는다.**" → 자동매매의 위험 경고.

→ 우리가 독립적으로 도출한 "덜 잃는 것이 핵심" 결론이 업계 상식과 부합한다.
