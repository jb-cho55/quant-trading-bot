# #6 — 성장주에 옮겨 심기: BTC 전략은 주식에 통할까?

BTC에서 워크포워드를 통과한 전략(추세필터·변동성 돌파)을 **개별 성장주(TSLA·NVDA)**
에 그대로 올리면 통할까? 모의투자로 가기 전, "종목을 바꿔도 살아남는가"를 확인했다.

## 코드

| 파일 | 내용 |
|---|---|
| `stock_validation.py` | BTC용 추세필터·변동성 돌파를 TSLA/NVDA에 워크포워드 재검증 |
| `meanreversion_tournament.py` | 정반대 철학(평균회귀): RSI·볼린저·눌림목 매수 토너먼트 |
| `momentum_rotation.py` | 대형 기술주 8종에서 모멘텀 상위 3종 순환 보유(분산+추세) |

## 핵심 결과 — "전략은 종목을 탄다"

1. **추세필터·변동성 돌파는 성장주에서 실패했다.** 둘 다 '하락 회피형'이라
   현금으로 빠져 있는 동안 성장주의 큰 상승을 놓쳤다 — BTC에서 좋았던 게
   TSLA/NVDA에선 오히려 독.
2. **평균회귀(눌림목 매수)** 로 방향을 틀어봤지만, 개별 종목은 변동이 커서
   "비슷하게 벌되 덜 잃기"가 깔끔히 성립하지 않았다.
3. **모멘텀 로테이션** — 한 종목에 베팅하지 않고 *바스켓에서 강한 종목만 순환 + 약하면 현금*
   = 개별 종목 리스크를 분산. 성장주 영역에서 가장 합리적인 형태였다.

> **교훈:** 한 자산에서 검증된 전략도 다른 자산에선 무너질 수 있다.
> 개별 종목 베팅보다 **분산(바스켓·자산배분)** 이 훨씬 견고하다 →
